# concessionnaire
Travail de fin d'année bac 2 JAVA - Concessionnaire d'occasion 

La base de donnée se nomme : cardealer
Le mot de passe est : root

Le projet est composé de 11 packages : 
- model : contient les modèles de données
- dataAccess : contient les classes de gestion de la base de données
- business : contient les classes de logique métier
- controller : contient les classes de contrôle de l'application
- view : contient les classes de l'interface utilisateur
- view --> utils : contient les classes utilitaires
- view --> components : contient les classes de composants graphiques
- view --> panels : contient les classes de panels graphiques
- SQL : contient les scripts SQL pour la création de la base de données et des tables
- config : contient les classes de configuration de l'application (static final variables)
- exceptions : contient les classes d'exceptions personnalisées
